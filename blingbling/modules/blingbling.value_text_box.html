Module blingbling.value_text_box

A text box that can display value and text with colors.

set_values_text_color (vt_box, table)
Define the value text color depending on the limit given ( if value >= limit, we apply the color). my_fs_root:set_values_text_color(table) -->{ {"#rrggbbaa", limit 1}, {"#rrggbbaa", limit 2}} By default the color is default_text_color(another example: {{"#88aa00ff",0},{"#d4aa00ff", 0.5},{"#d45500ff",0.75}})

Parameters

  • vt_box: the value text box
  • table: table like { {"#rrggbbaa", limit 1}, {"#rrggbbaa", limit 2}}
